# Fast FinTech Dashboard - Suraj Gupta

A high-performance Finance Dashboard built for the Frontend Developer Internship assignment. It provides a clean interface for tracking transactions, visualizing spending patterns, and simulating role-based access.

## ✨ Key Features
- **📊 Real-time Analytics:** Automated calculation of Total Balance, Income, and Expenses.
- **🛡️ Role-Based UI (RBAC Simulation):** - **Admin:** Can add new transactions and view all data.
  - **Viewer:** Read-only access with restricted actions.
- **📈 Interactive Visualizations:** - Area Charts for balance trends over time.
  - Donut Charts for categorical spending breakdown.
- **🌓 Dark Mode:** Fully functional dark/light mode toggle for better UX.
- **📱 Responsive Design:** Seamless experience across Mobile, Tablet, and Desktop.
- **🔍 Advanced Filtering:** Filter transactions by type (Income/Expense) instantly.

## 🛠️ Technical Stack
- **Library:** React 18
- **Styling:** Tailwind CSS
- **State Management:** React Context API (Chosen for lightweight and efficient global state)
- **Charts:** Recharts
- **Icons:** React Icons
